[what are cookies?]
Tokens are a type of non-full access alts
They come in a .txt file and can, depending on quality, last months
Below is a guide on how to use them
[warning]
If you buy cookies, you might receive them in a .zip file. Be sure to extract it
to have each individual file
[Use case]
Option 1: Your client supports direct cookie use
Option 2: Using forge 1.8.9
Option 3: Using a web browser (universal)
[1- direct cookies support]
Some clients (like rise, astralis, etc) support direct use of cookies
In their alt manager, simply go in cookies, select the file and press login
If it fails or the client doesn't support it, don't worry -> option 3
[2- using forge for direct cookie use]
Download the mod at the end of this page
On the main page, press the small button next to "quit game"
Press "add" then "cookie" and select the file; It should now be added
[3- using a web browser]
The following steps require you to use firefox or a firefox-based browser
If you use another browser, consider installing firefox and using it only for login
Install the extension here
While on the microsoft login page, click the extension -> search cookies
In the windows it opened, press the import button (bottom left) and click "restore cookies from a file"
Select the file then reload your login page; The account should now appear
If it asks for 2FA, try to first login to minecraft.net
