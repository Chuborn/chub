[hypixel bypasses]
Raven bsv2 is a client featuring pretty impressive bypasses for a free client. This includes things such as a sword noslow, something prediction anticheat showed us should not be taken for granted. It also features a pretty fast telly scaffold, one of the only free client and the most reliable of these free clients. I wouldn't be shocked if this was a paid client, so you really can't complain. If you are to use it as a ghost client, it does also feature the basics and all the visuals you can dream of in a lightweight mod form, tho I personally don't use it for that
[multiserver bypasses]
Raven bsv2 has not been made with multiserver in mind, but being a ghost - blatant client and the presence of scripting and a ton of modules means it is actually possible to use it on server that use anticheats like GRIM, but it still is pretty limited
[personal opinion]
Honestly, I think raven bsv2 is the best free client there is in every way. The bypasses are there even if it might be discontinued while the visuals and the client feel are good, unlike other free alternative that feel clunky, ugly and hard / annoying to use (I see you Liquidbounce).
