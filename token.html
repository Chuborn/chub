[what are tokens?]
Tokens are a type of non-full access alts that last 24h
[how to format tokens]
Below is how a token look when received from a shop
eyiojef(random shit)|accountusername|accountuuid
Only the beginning is needed, you must remove |accountusername|accountuuid
[how to login]
Option 1: Your client supports it - Just paste the token and hit login
Option 2: You use forge/fabric
Mods for token login are available for Minecraft 1.8.9 and 1.21.5
If your client does not support token login or you use any other version, token can't be used
[for windows - other launchers]
Navigate to the settings menu
Find the java tab / menu
Select or manually input "placeholder" into "java path"
Note that this path is for the provided download and may differ
[for linux - prism]
Click on edit
Go to settings -> Java
Check "java installation" if not checked
Go to "adoptium" -> Java 8 -> Select the latest version and download it
Go to "auto detect" and find the item that starts with "1.8" and has the word "Eclipse" in the file path
Launch the instance as usual
