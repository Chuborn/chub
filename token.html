[what are tokens?]
Tokens are a type of non-full access alts that last 24h
[how to format tokens]
Below is how a token might when received from a shop*
eyiojef(random shit)|accountusername|accountuuid
Only the beginning is needed, you must remove |accountusername|accountuuid
Not all shop use this format. Simply make sure that what you copy starts with "ey" and doesnt contain |accountusername|accountuuid or any other shit not part of the token
This process can be automated with this website, but this only works with some format and is not always needed.
*some shop preformat the accounts
[how to login]
Option 1: Your client supports it - Just paste the token and hit login
Option 2: You use forge/fabric
Mods for token login are available for Minecraft 1.8.9 and 1.21.5
Option 3: You use any other version
Thanks to ZZXGP, Token login is also avaiable for Lunar Client Launcher. This means token ca be used on any minecraft versions
These are currently all the known safe way to use token accounts
