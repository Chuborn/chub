[what are tokens?]
Tokens are a type of non-full access alts that last 24h
Below is a guide on how to use token for each situation
[1- how to format tokens]
Token are a long string of letters that start with "ey".
When you use a shop, you can receive it in multiple form
Case 1: They sometime come preformated (nicealt/zz/etc) but can also be unformated -> Proceed to step 2
Case 2: They can come with |accountusername|accountuuid at the end; it must be removed manually or with this website -> Proceed to step 2
[2- how to login]
Option 1: Your client supports it
Option 2: You use forge / fabric (1.8.9 or 1.21.x)
Option 3: You use Lunar Client or any other version of minecraft
[option 1 - client supports it]
Some clients (like rise) support token login natively
Just paste the token in "session" or "refresh" token
[option 2 - forge / fabric]
Download the mod for your minecraft version below
On the server selector, a box on the top right allows you to paste and login
[option 3 - lunar client / other mc versions]
Download the lunar token login tool below
Run it while Lunar Launcher is closed
Paste your token and press enter; you can now use any version from Lunar Launcher
