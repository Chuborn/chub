[fix authentification]
Following the expiration of java certificates, user can meet the "authentification server are down" error
[requirements]
Step 1: Donwload Java Eclipse Adoptium 8 on the web or with the link below and install it:
download for windows
download for linux -> See linux tutorial below
[for windows - official launcher]
Navigate to the "Installations" tab
Click on your affected game profile
Click on more options to reveal additional settings
Paste "C:\Program Files\Eclipse Adoptium\jdk-8.0.432.6-hotspot\bin\javaw.exe" in "java executable"
Note that this path is for the provided download and may differ
[for windows - other launchers]
Navigate to the settings menu
Find the java tab / menu
Select or manually input "C:\Program Files\Eclipse Adoptium\jdk-8.0.432.6-hotspot\bin\javaw.exe" into "java path"
Note that this path is for the provided download and may differ
[for linux - prism]
Click on edit
Go to settings -> Java
Check "java installation" if not checked
Go to "adoptium" -> Java 8 -> Select the latest version and download it
Go to "auto detect" and find the item that starts with "1.8" and has the word "Eclipse" in the file path
Launch the instance as usual
